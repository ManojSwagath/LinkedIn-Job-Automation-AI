          LINKEDIN JOB AUTOMATION WITH AI - SYSTEM ARCHITECTURE
================================================================================

PROJECT OVERVIEW
================================================================================
An autonomous AI agent system that discovers and applies to LinkedIn jobs 
automatically on behalf of users. The system combines web automation, 
artificial intelligence, and modern web technologies to streamline the 
job application process.

    * Hashing: bcrypt
    * Salting: Automatic
    * Validation: Passlib
  
  - dependencies.py: FastAPI dependency injectors
    * get_current_user(): JWT validation
    * get_current_active_user(): Active user check
  
  - validators.py: Input validation
    * Email format validation
    * Password strength rules
    * Phone number formatting

================================================================================
                        SECURITY CONSIDERATIONS

AUTHENTICATION & AUTHORIZATION
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1. JWT Tokens
   - Algorithm: HS256
   - Expiry: 30 minutes (access), 7 days (refresh)
   - Storage: Frontend localStorage
   - Transmission: Authorization: Bearer <token>

2. Password Security
   - Hashing: bcrypt with automatic salting
   - Minimum length: 8 characters
   - Strength validation: Required

3. LinkedIn Credentials
   - Encryption: Required for storage
   - Never logged or exposed in responses
   - Stored in separate credentials table

DATA PROTECTION
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1. HTTPS Only
   - All API calls encrypted in transit
   - Render provides automatic SSL
   - Vercel enforces HTTPS

2. CORS Configuration
   - Whitelist only Vercel frontend domain
   - No wildcard origins in production
   - Credentials enabled for cookie-based auth

3. Input Validation
   - Pydantic models for all endpoints
   - File upload size limits (10 MB)
   - File type restrictions (PDF, DOCX only)
   - SQL injection prevention (SQLAlchemy ORM)

4. Rate Limiting
   - LinkedIn: Max applications per day
   - API: Rate limiting per user
   - Anti-bot delays in automation

ANTI-DETECTION MEASURES
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1. Human-like Behavior
   - Random typing speeds
   - Mouse movement simulation

================================================================================
                        TROUBLESHOOTING GUIDE
================================================================================

COMMON ISSUES & SOLUTIONS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Issue: Vercel 500 MB Lambda size limit error
Solution:
  - Ensure vercel.json exists in repo root
  - Verify .vercelignore excludes backend/
  - Check that vercel.json only builds frontend/lovable
  - Backend should NEVER be deployed to Vercel

Issue: Playwright browser not found on Render
Solution:
  - Verify build.sh runs: playwright install chromium
  - Check PLAYWRIGHT_BROWSERS_PATH in env vars
  - Ensure Render plan supports required memory

Issue: CORS errors in production
Solution:
  - Add Vercel frontend URL to CORS_ORIGINS env var
  - Format: https://yourapp.vercel.app (no trailing slash)
  - Restart Render service after env var change

Issue: Database connection refused
Solution:
  - Verify DATABASE_URL format: postgresql://user:pass@host:5432/db
  - Check Supabase dashboard for correct credentials
  - Test connection from Render shell

Issue: LinkedIn login fails
Solution:
  - Verify LINKEDIN_EMAIL and LINKEDIN_PASSWORD
  - Check if 2FA is enabled (requires manual intervention)
  - Review browser screenshots for captcha
  - Reduce automation frequency (anti-bot)

Issue: OpenAI API rate limit
Solution:
  - Implement exponential backoff
  - Cache LLM responses
  - Switch to Gemini (higher free tier)
  - Upgrade OpenAI tier

Issue: Application submission fails
Solution:
  - Enable dry_run mode first to test
  - Check for form changes on LinkedIn
  - Review error logs in applications table
  - Verify resume file upload works
