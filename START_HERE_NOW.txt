⏳ 3 STEPS TO RUN (5 minutes)
═══════════════════════════════════════════════════════════

Step 1: Configure Credentials (2 minutes) ⚙️
────────────────────────────────────────────
Edit .env file:

   code .env
   # or
   nano .env

Update these 2 fields:

   OPENAI_API_KEY=sk-proj-YOUR_ACTUAL_KEY_HERE
   LINKEDIN_EMAIL=your-email@example.com
   LINKEDIN_PASSWORD=your-actual-password

Get OpenAI key: https://platform.openai.com/api-keys

────────────────────────────────────────────

Step 2: Add Your Resume (1 minute) 📄
────────────────────────────────────────────
   mkdir -p data/resumes
   cp ~/Downloads/your_resume.pdf data/resumes/

Supported: PDF, DOCX, TXT

────────────────────────────────────────────

Step 3: Run the System! (< 1 minute to start) 🚀
────────────────────────────────────────────
   ./START_PROJECT.sh

Choose Option 2: "Run Direct Workflow"

Then provide:
   • Job keywords (e.g., "Python Developer")
   • Location (e.g., "Remote", "San Francisco")
   • Max jobs to process (5-10 recommended)

🔧 TROUBLESHOOTING
═══════════════════════════════════════════════════════════

Problem: OpenAI API key invalid
────────────────────────────────────────────
Solution:
   1. Go to https://platform.openai.com/api-keys
   2. Create new key (starts with sk-proj-)
   3. Update .env: OPENAI_API_KEY=sk-proj-...
   4. Ensure account has credits

────────────────────────────────────────────

Problem: LinkedIn login failed
────────────────────────────────────────────
Solution:
   1. Verify email/password in .env
   2. Try manual login first
   3. Disable 2FA temporarily
   4. Set HEADLESS_BROWSER=false to see browser

────────────────────────────────────────────

Problem: No resume found
────────────────────────────────────────────
Solution:
   mkdir -p data/resumes
   cp ~/Downloads/your_resume.pdf data/resumes/
   ls data/resumes/  # Verify it's there

────────────────────────────────────────────

Problem: Permission denied
────────────────────────────────────────────
Solution:
   chmod +x START_PROJECT.sh

────────────────────────────────────────────

More help: See RUN_PROJECT_GUIDE.md
